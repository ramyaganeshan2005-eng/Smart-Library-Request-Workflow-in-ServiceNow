# Smart Library Workflow Project

## Overview
The Smart Library is a **web-based library management system** that simulates a ServiceNow-style workflow using **Node.js, Express, and a simple frontend**.  
It allows **students** to request books, **librarians** to approve requests, and automatically updates the **book status**. Reports can show the most borrowed books.

---

## Features

- View all books and their status (Available / Issued / Lost)
- Students can create **borrow requests**
- Librarians can **approve requests**
- Book status updates automatically after approval
- Prevents requesting already issued books
- Report API for **Most Borrowed Books**
- Simple frontend interface
